### Goal

To develop admin panel for Vanilla Junction Project using React (Vite), TailwindCSS, ShadCN, TypeScript and securely connecting it with the backend.

### Requirements

1. Admin
   1. Can log in
   2. List ice-cream items
   3. Update quantities
   4. Update ice-cream details
   5. Add delivery persons
   6. View all users
   7. View all Orders
   8. Manually Modify the order status.
2. Delivery Person
   1. Can log in
   2. Check new deliveries
   3. Accept a delivery
   4. Request a delivery OTP
   5. Verify the OTP.
